# 📚 توثيق نظام التثبيت — Mini CRM

> **اللغة:** العربية  
> **النوع:** توثيق تسليم للمطور أو الفريق التقني  
> **النظام:** Laravel — نظام التثبيت الإنتاجي ذو الاستخدام الواحد  

---

## وصف المشروع

هذا المجلد يحتوي على التوثيق الكامل لنظام التثبيت الآمن المُضمَّن في مشروع **Mini CRM** المبني على Laravel.

النظام مصمَّم لتمكين نشر التطبيق على أي استضافة مشتركة أو سيرفر VPS بواجهة ويب بسيطة وآمنة، دون الحاجة إلى الوصول عبر SSH أو تنفيذ أوامر يدوية.
